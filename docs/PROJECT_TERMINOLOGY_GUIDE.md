# 專案用語指南

## 概述

技術術語保持英文，中文說明使用台灣慣用語，避免使用中國或其他地區常見用語。

## 基本原則

- 技術標準、API、程式語言名稱維持英文原文（以反引號 ` 保留）。
- 中文僅作為說明與使用情境輔助，不進行翻譯或在地化改寫。
- 同一語意僅保留一組專案用語，避免混用。

## 共享與共用

| 用語   | 英文對應 | 適用層級        | 專案使用原則                 | 範例                   |
| ------ | -------- | --------------- | ---------------------------- | ---------------------- |
| `共享` | Sharing  | UI / 使用者行為 | 表示「分享出去」的動作或功能 | QR Code 分享、分享連結 |
| `共用` | Shared   | 系統 / 程式資源 | 表示多模組或多元件共同使用   | 共用記憶體、共用模組   |

**統一原則**

- 與使用者互動、功能操作相關 → 使用「共享」
- 與系統、程式、資源結構相關 → 使用「共用」
