# 專案用語指南

## 概述

技術術語保持英文，中文說明使用台灣習慣用語，避免中國或其他地區用語。

## 基本規則

### 技術術語

| 歧異用語                         | 專案用語                               | 適用場景     | 範例         |
| -------------------------------- | -------------------------------------- | ------------ | ------------ |
| 索引資料庫、網路通訊端、爪哇腳本 | `IndexedDB`、`WebSocket`、`JavaScript` | 程式設計術語 | 保持英文原文 |

### 在地認可用語

| 術語                                   | 狀態         | 說明                                          |
| -------------------------------------- | ------------ | --------------------------------------------- |
| **響應式設計 (Responsive Web Design)** | **在地認可** | 台灣 UI/UX 領域之標準慣用語，教科書與業界通用 |
| **自適應 (Adaptive)**                  | **具歧異性** | 台灣較常用於演算法領域，UI 設計較少使用       |

### 常見修正

#### 數量詞

| 歧異用語 | 專案用語 | 適用場景 | 範例         |
| -------- | -------- | -------- | ------------ |
| `批量`   | `批次`   | 資料處理 | 批次處理資料 |

#### 電腦/技術含義

| -------- | -------- | -------- | -------------------- |

#### 地理/時區含義

| 歧異用語 | 專案用語 | 適用場景 | 範例               |
| -------- | -------- | -------- | ------------------ |
| `本機`   | `本地`   | 地理位置 | 本地時間、本地語言 |

## 特別說明：共享 vs 共用

- **共享 (Sharing)**：UI 功能，分享資料、分享連結等
  - 範例：QR Code 共享 → QR Code 分享

- **共用 (Shared)**：程式資源，共用記憶體、共用資源等
  - 範例：共享記憶體 → 共用記憶體
